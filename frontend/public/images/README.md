# Barangay Assets

## Logo Files

Place your barangay logo files in this directory:

- `barangay-logo.png` - Main barangay logo (recommended: 512x512px)
- `barangay-logo-sm.png` - Small logo for sidebar (recommended: 128x128px)
- `barangay-seal.png` - Official seal (if different from logo)

## Usage

In your React components:

```jsx
<img src="/images/barangay-logo.png" alt="Barangay Logo" />
```

## Supported Formats

- PNG (recommended for logos with transparency)
- SVG (best for scalability)
- JPG (for photos)
